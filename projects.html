Practice and feedback on software testing

Software testing is an important self-regulatory skill involved in software development. Unfortunately, many students practice software testing as an afterthought, and face difficulties with testing that follow them into the work force. I'm interested in the following research directions:

  • Observing students' current software testing practices. How do students practice testing? How do we want them to practice testing? [SIGCSE '19']
  • The use of mutation analysis as a device to provide students with feedback and practice with software testing [under review]

Assessing and improving self-regulatory skills in software development

Novice software developers often face self-regulatory challenges when faced with large or complex programming projects (relative to what they have seen in the past). Successful completion of these projects requires declarative knowledge of how to write programs (which they typically have) as well as procedural knowledge of how to engineer software (which they typically don't have).

Changing the way this material is learned requires changing how it is taught, practiced, and assessed. To that end, we built DevEventTracker to collect high-resolution log data from Eclipse.

  • Using this data, we developed and qualitatively evaluated quantitative measurements of students' time management practices on long-running programming projects [ITiCSE '17]
  • We found that students' time management practices had significant impacts on their project performance and timeliness, even after controlling for individual student abilities [ICER '17]

CodeWorkout—Drill-and-practice programming exercises for novices

CodeWorkout is an online drill-and-practice system for people learning a programming language for the first time. It is free, open-source, and currently serves thousands of users at Virginia Tech and other universities.

See the project on GitHub.

  • We used CodeWorkout in our CS 1 course and explored the effects of voluntary practice of programming assignments on exam performance, controlling for individual student abilities [CompEd '19]
